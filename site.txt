Страницы:
1) Главная:
1. Лендинг - если пользователь не авторизирован:
2. Панель инструментов - если пользователь авторизирован:
1. Аналитика - в ней будут различные метрики, графики и сравнения. (Например, кол-во просмотров карточки мастера, соотношение просмотрели/написал)
2. Уровни и достижения - здесь будет система баллов. (Например, за выполнение каких-либо заданий)
3. Финансы - та же аналитика, только с финансами, помимо чего здесь можно будет управлять системой развития аккаунта (продвижение на платформе, покупка какого-нибудь доп. функционала)
4. Менеджер задач - простой таск-трекер по типу trello, только внутри платформы (задачи, в работе, выполненные и тд)
5. Расписание - график с расписанием мастера, тут в принципе должно быть понятно.
6. Партнёрская программа - за приведение новых клиентов, будут начисляться бонусы (Например, за приглашение человека в платформу по ссылке, пользователь получит 50 баллов, которые может потратить на развитие своего аккаутна)

2) Услуги:
1. Фильтр для поиска услуг
2. Список услуг (карточки услуг с указанием мастера)
3. Возможность открыть карточку мастера по нажатию на услугу.

3) Запросы:
1. Список запросов определенных услуг от пользователей
2. Запросы имеют срок, после которого они удаляются
3. Возможность пользователю создать запрос на кнопку "+ Создать запрос" 

4) Рейтинг:
Различные рейтинги: 
1. по количеству отзывов, 
2. по качеству, 
3. по количеству выполненных услуг

5) Профиль:
1. Аватарка
2. Основная информация (Имя Фамилия/Название)
3. Биография/о себе
4. Услуги, цены
5. Возможность редактирования владельцем

6) Настройки:
2. Безопасность (всё связанное с авторизацией)
3. Приватность

7) Чаты:
1. Возможность написать человеку по username
2. Возможность перейти в беседу
3. Возможность удаления чатов
4. Возможность удаления/редактирования сообщения

Основное:
1. Профиль
2. Услуги
3. Чат
